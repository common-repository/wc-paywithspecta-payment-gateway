== Description ==

This is a PayWithSpecta payment gateway for WooCommerce by Acesys Solutions.

PaywithSpecta was designed to help merchants to increase sales and enable consumers to enjoy the good things of life without necessarily killing themselves to pay.

To signup for a PayWithSpecta Merchant account visit their website by clicking [here](https://paywithspecta.com/merchant)

Payment Gateway for PayWithSpecta allows your buyers to pay instalmentally to Specta Loans while you receive your payments in full instantly.

With this Payment Gateway for PayWithSpecta plugin, you will be able to boost sales getting paid instantly while your customers enjoy flexible payment options


= Note =

This plugin is meant to be used by merchants in Nigeria only and only applies to purchases above 20,000 naira.

= Plugin Features =

*   __Minimum configuration required__ just your merchant ID and PayWithSpecta API key.
*   __Seamless integration__ into the WooCommerce checkout page.

== Installation ==

= Automatic Installation =
* 	Login to your WordPress Admin area
* 	Go to "Plugins > Add New" from the left hand menu
* 	In the search box type __Acesys PayWithSpecta Payment Gateway__
*	From the search result you will see __Acesys PayWithSpecta Payment Gateway__ click on __Install Now__ to install the plugin
*	A popup window will ask you to confirm your wish to install the Plugin.
*	After installation, activate the plugin.
* 	Open the settings page for WooCommerce and click the "Payments" tab.
* 	Click on the __PayWithSpecta__ link from the available Payment Options
*	Configure your __Payment Gateway for PayWithSpecta__ settings. See below for details.

= Manual Installation =
1. 	Download the plugin zip file
2. 	Login to your WordPress Admin. Click on "Plugins > Add New" from the left hand menu.
3.  Click on the "Upload" option, then click "Choose File" to select the zip file from your computer. Once selected, press "OK" and press the "Install Now" button.
4.  Activate the plugin.
5. 	Open the settings page for WooCommerce and click the "Payments" tab.
6. 	Click on the __PayWithSpecta__ link from the available Payment Options
7.	Configure your __Payment Gateway for PayWithSpecta__ settings. See below for details.



= Configure the plugin =
To configure the plugin, go to __WooCommerce > Settings__ from the left hand menu, then click __Payments__ from the top tab. You will see __Paystack__ as part of the available Checkout Options. Click on it to configure the payment gateway.

* __Enable/Disable__ - check the box to enable PayWithSpecta Payment Gateway.
* __Title__ - allows you to determine what your customers will see this payment option as on the checkout page.
* __Description__ - controls the message that appears under the payment fields on the checkout page.
* __Minimum Order Amount__ - This sets the minimum purchase amount for which PayWithSpecta option will be enable during checkout. The minimum is 20,000 naira.
* __Test Mode__ - Check to enable test mode. Test mode enables you to test payments before going live. If you ready to start receving real payment on your site, kindly uncheck this.
* __Test API Key__ - Enter TEST_API_KEY.
* __Live API Key__ - Enter your Live API Key here. Get your API key from your PayWithSpecta Merchant account dashboard under Settings.
* Click on __Save Changes__ for the changes you made to be effected.
